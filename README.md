# 🖥️ CodeAlpha Frontend Development Internship

> Internship projects completed as part of the **CodeAlpha Frontend Development Internship Program**.  
> This repository contains hands-on frontend projects built using HTML, CSS, and JavaScript.

---

## 📁 Projects Included

---

### ✅ Task 1 — Image Gallery

A responsive image gallery with smooth navigation and visual effects.

**Features:**
- Grid-based layout using HTML & CSS
- Next / Previous navigation buttons
- Lightbox view for full-size image preview
- Hover effects and smooth CSS transitions
- Fully responsive design for mobile, tablet, and desktop

**Bonus Features:**
- Image filters (e.g., grayscale, brightness)
- Category-based image sorting and filtering

---

### ✅ Task 2 — Calculator

A clean, functional calculator supporting all basic arithmetic operations.

**Features:**
- Basic arithmetic: Addition, Subtraction, Multiplication, Division
- Real-time result display
- Clear / reset screen functionality
- Handles edge cases (division by zero, chained operations)
- Buttons and display screen UI

**Bonus Features:**
- Keyboard input support
- Enhanced UI styling and animations

---

### ✅ Task 3 — Personal Portfolio Website

A personal portfolio showcasing skills, projects, and contact information.

**Features:**
- Sections: About, Skills, Projects, Resume, Contact
- Smooth scrolling between sections
- CSS animations and hover effects
- Clean, responsive layout for all screen sizes

**Bonus Features:**
- Deployed on GitHub Pages / Netlify

---

### ✅ Task 4 — Music Player

An interactive music player with full audio control functionality.

**Features:**
- Play, Pause, Next, Previous track controls
- Displays song title, artist name, and duration
- Progress bar with seek functionality
- Volume control slider

**Bonus Features:**
- Playlist support
- Autoplay on track end
