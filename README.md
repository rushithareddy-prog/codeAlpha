# 🖥️ CodeAlpha Frontend Development Internship

> Internship projects completed as part of the **CodeAlpha Frontend Development Internship Program**.
> This repository contains hands-on frontend projects built using HTML, CSS, and JavaScript.

---

## 📁 Projects Included

---

### 🖼️ Task 1 — Image Gallery

A responsive image gallery with smooth navigation and visual effects.

**Features:**

* Grid-based layout using HTML & CSS
* Next / Previous navigation buttons
* Lightbox view for full-size image preview
* Hover effects and smooth CSS transitions
* Fully responsive design

**Enhanced Features:**

* Image filters (grayscale, brightness, etc.)
* Category-based sorting and filtering

---

### 🧮 Task 2 — Calculator

A clean and functional calculator supporting basic arithmetic operations.

**Features:**

* Addition, Subtraction, Multiplication, Division
* Real-time result display
* Clear/reset functionality
* Handles edge cases (e.g., division by zero)
* Interactive button-based UI

**Enhanced Features:**

* Keyboard input support
* Improved UI styling and animations

---

### 🌐 Task 3 — Personal Portfolio Website

A personal portfolio website showcasing skills and projects.

**Features:**

* Sections: About, Skills, Projects, Resume, Contact
* Smooth scrolling navigation
* Responsive layout for all devices
* Clean design with animations and hover effects

**Enhanced Features:**

* Deployment using GitHub Pages / Netlify

---

### 🎵 Task 4 — Music Player

An interactive music player with audio control features.

**Features:**

* Play, Pause, Next, Previous controls
* Displays song title and artist
* Progress bar with seek functionality
* Volume control

**Enhanced Features:**

* Playlist support
* Auto-play feature
